My journey!

  • Block coding!

    I started coding 3 years ago. My journey began by learning a coding method called "Block Coding". It is a very basic coding method which helped me understand the basics of coding or what it is. What exactly is block coding? Block coding is a method used in computer science and information theory to represent and transmit data in a structured and organized manner. It involves dividing data into fixed-size blocks or groups of bits, each containing a predefined number of binary digits (0s and 1s). These blocks are created with specific patterns and redundancy information, which helps in error detection and correction during data transmission or storage. So instead of typing in every single code, you can use blocks and drag and drop!

  • Text based programming language

    After block coding, I needed to go to the next level of coding which is text based programming.Unlike block coding, in text based we type in every single code so that the computer can understand.I began with a coding website called "CodePen" and soon started understanding how this works. Basic Structure of Coding!

Variables!

A variable is a temporary container to store information, it is a named location in computer memory where varying data, like numbers and characters, can be stored and manipulated during the execution of the program. Variables are used to help coders use the information in the program.

Conditional Statements!

A type of coding instruction used to compare values and express and make decisions. A conditional statement tells a program to execute an action depending on whether a condition is true or false. It is often represented as an if-then or if-then-else statement.

Functions!

Functions are "self contained" modules of code that accomplish a specific task. Functions usually "take in" data, process it, and "return" a result. Once a function is written, it can be used over and over again. Functions can be "called" from the inside of other functions.
